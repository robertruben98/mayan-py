# mayan-py

Python client for the Mayan Finance API — cross-chain swap and bridge across Solana, EVM and Sui. Sync + async, Pydantic v2.

## Install

```bash
pip install mayan-py
```

The distribution is `mayan-py`; the import package is `mayan`.

## Quick start

```python
import mayan
```

See the [API reference](api.md) for the full surface.
